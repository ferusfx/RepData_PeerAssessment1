  1. Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc.
act <- act %>% 
        group_by(interval) %>% 
        mutate(steps_mean_int = mean(steps, na.rm = T))

for(i in 1:nrow(act)){
        if(is.na(act[i,1]) == TRUE){
                act[i,1] <- act[i,4] # If NA, set to mean of corresponding 5-minute interval.
        }
}

Because we took the mean of the steps for each interval as a replacement for the NAs the difference between both parts of the assignments in the end result is almost non-existent. Naturally, the median differs slightly (by 1). Imputing missing data with other values than the mean would result in more significant differences.

Are there differences in activity patterns between weekdays and weekends?

  1. Create a new factor variable in the dataset with two levels – “weekday” and “weekend” indicating whether a given date is a weekday or weekend day.
Sys.setlocale("LC_TIME", "English")
## [1] "English_United States.1252"
day <- weekdays(act_new$date)
day_type <- vector()
for(i in 1:nrow(act_new)) {
    if(day[i] == "Saturday" | day[i] == "Sunday"){
        day_type[i] <- "Weekend"
    }else{
        day_type[i] <- "Weekday"
    }
}
act_new$day_type <- day_type
act_new$day_type <- factor(act_new$day_type)
